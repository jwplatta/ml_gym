## Numpy

- integer indexing
- boolean indexing
	- using a mask to select or set values
- functions
	- universal functions, e.g. `np.sqrt`, `np.log`
		- they apply element wise to each element in the array
		- operate on multiple arrays
	- note direction of operation
		- np.mean(0) - mean across rows
		- np.mean(1) - mean across columns
	- math and stats functions
	- boolean array operators: `np`
	- `arr1.sort()`, `np.unique(arr1)`
- linear algebra
	- better done in numpy than pandas
