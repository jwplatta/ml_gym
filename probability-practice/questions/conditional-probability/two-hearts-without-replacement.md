# Two Hearts Without Replacement
## Question
A card is drawn from a standard 52-card deck, then a second card is drawn without replacement.

What is the probability that the first card is a heart and the second card is also a heart?

Write the answer using the multiplication rule.
## Solution
P(first heart) = 13/52 = 1/4

Given that the first card was a heart:
P(second heart | first heart) = 12/51 = 4/17

So:
P(heart then heart) = (13/52) × (12/51) = (1/4) × (4/17) = 1/17

**Answer:** 1/17 ≈ 0.0588

This is a conditional probability problem because the second probability changes after the first draw.
