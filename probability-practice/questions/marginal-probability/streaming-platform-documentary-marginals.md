# Streaming Platform Documentary Marginals
## Question
A streaming platform classifies users by subscription tier and whether they watched a documentary this week.

|                | Watched Documentary (D) | Did Not Watch (¬D) | Total |
|----------------|--------------------------|--------------------|-------|
| Premium (P)    | 54                       | 36                 | 90    |
| Basic (¬P)     | 21                       | 39                 | 60    |
| **Total**      | **75**                   | **75**             | **150** |

Find:
1. P(P), the probability a randomly selected user is a Premium subscriber
2. P(D), the probability a randomly selected user watched a documentary
## Solution
The total number of users is 150.

P(P) = 90/150 = 3/5 = 0.60

P(D) = 75/150 = 1/2 = 0.50

**Answers:**
- P(P) = 0.60
- P(D) = 0.50

These are both marginal probabilities because they come from totals across an entire row or column.
